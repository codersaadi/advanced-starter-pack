## Planning Our Simple Bot: The Greeter

Let's design a bot with a minimal task: responding to basic greetings and a couple of simple questions.

**Goal:**
*   Recognize greetings like "hello", "hi", "hey".
*   Respond to a question about its name.
*   Respond to a question about what it can do.
*   Handle unrecognized input gracefully.
*   Exit when the user types "bye".

**Approach:** Rule-Based Keyword Matching. We'll check if specific keywords exist in the user's input and trigger a predefined response.

## The Core Logic: Input -> Process -> Output

Every chatbot, simple or complex, follows a fundamental loop:

1.  **Get Input:** Receive text from the user.
2.  **Process Input:** Analyze the text to understand intent (or in our simple case, look for keywords).
3.  **Generate Output:** Formulate and display a response.
4.  **Loop/Exit:** Repeat the process or terminate if requested.

## Let's Code! A Simple Python Implementation

We'll use basic Python string methods and conditional logic (`if`/`elif`/`else`).

```python title="simple_chatbot.py"
# A very basic rule-based chatbot in Python

def get_response(user_input):
  """Generates a response based on keywords in the user input."""
  # Convert input to lowercase for case-insensitive matching
  lowered_input = user_input.lower()

  # Define rules (keyword -> response)
  if "hello" in lowered_input or "hi" in lowered_input or "hey" in lowered_input:
    return "Hello there! How can I help you today?"
  elif "how are you" in lowered_input:
    return "I'm just a bunch of code, but I'm running smoothly! Thanks for asking."
  elif "your name" in lowered_input:
    return "I don't have a name yet, you can call me SimpleBot."
  elif "what can you do" in lowered_input or "help" in lowered_input:
    return "I can respond to greetings, tell you my 'name', and understand 'bye' to exit."
  elif "bye" in lowered_input or "exit" in lowered_input:
    return "Goodbye! Have a great day."
  else:
    # Default response if no keywords match
    return "Sorry, I didn't quite understand that. Can you rephrase or ask something else?"

# --- Main Chat Loop ---
print("SimpleBot: Hi! I'm a basic chatbot. Type 'bye' to exit.")

while True:
  user_message = input("You: ")
  response = get_response(user_message)
  print(f"SimpleBot: {response}")

  # Check if the user wants to exit
  if "bye" in user_message.lower() or "exit" in user_message.lower():
    break # Exit the loop

print("\nSimpleBot: Shutting down...")
```

## The Obvious Limitations (Why AI is Needed)

This simple bot highlights why AI, especially NLP, is crucial for real-world chatbots:

1.  **No Context:** It doesn't remember previous parts of the conversation. Each input is treated independently.
2.  **Brittle Matching:** It only understands exact keywords. "Hi there" works, but "Yo" doesn't. "What do you do?" fails, but "what can you do" works.
3.  **No Real Understanding:** It's just pattern matching, not comprehending meaning or intent. Asking "Can you tell me your designation?" instead of "What is your name?" would fail.
4.  **No Learning:** It cannot improve or adapt based on interactions.
5.  **Scalability:** Adding hundreds or thousands of rules manually becomes unmanageable.

## Taking the Next Step: Beyond Simple Rules

To overcome these limitations, you'd typically explore:

*   **Regular Expressions (Regex):** More flexible pattern matching than simple keyword checks. Python's `re` module is powerful here.
*   **Basic NLP Libraries (`NLTK`, `spaCy`):** For tasks like:
    *   *Tokenization:* Splitting text into words or sentences.
    *   *Stemming/Lemmatization:* Reducing words to their root form ("running" -> "run").
    *   *Part-of-Speech Tagging:* Identifying nouns, verbs, adjectives, etc.
    *   *Named Entity Recognition (NER):* Finding names, dates, locations.
*   **Intent Recognition Models:** Using Machine Learning (like the conceptual example in our previous post) to classify the user's goal even with varied phrasing.
*   **Chatbot Frameworks (`Rasa`, `Microsoft Bot Framework`, `Dialogflow`):** These provide tools for managing dialogue flow, state (context), integrations, and often incorporate ML/NLP capabilities.

<Callout type="tip">
  Start by exploring Python's `re` module for more robust pattern matching, or dip your toes into `NLTK` for basic text processing tasks.
</Callout>
